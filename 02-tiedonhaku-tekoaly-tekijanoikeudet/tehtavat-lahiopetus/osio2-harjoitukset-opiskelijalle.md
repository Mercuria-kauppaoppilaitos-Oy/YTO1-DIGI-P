## Tehtävä 2.1 — Tiedonhaku ja lähdekritiikki

**Arviointi:** Hyväksytty / Hylätty  
**Suoritusmuoto:** Pari- tai yksilötyö  
**Palautus:** Täytetty pohja (Word/PDF) eKampukseen

---

### Tilanne

Olet töissä kaupan alalla ja saat johtajaltasi sähköpostin:

> *"Kuulin, että nuorten ostokäyttäytyminen on muuttunut radikaalisti — he ostavat enää lähes kaiken netistä eivätkä käy kivijalkakaupoissa. Onko tässä perää? Tarvitsen luotettavan vastauksen ennen ensi viikon palaveria."*

Tehtäväsi on selvittää asia **neljästä eri lähteestä** ja raportoida johtajallesi.

---

### Vaihe 1 — Tiedonhaku (neljä lähdettä)

Etsi tietoa nuorten ostokäyttäytymisestä tai verkkokaupan kasvusta Suomessa. Käytä **jokaista alla olevaa lähdettä** ja kirjaa löytämäsi tieto taulukkoon.

| # | Lähdetyyppi | Käytä tätä | Mitä löysit? (kirjaa lyhyesti) |
|---|---|---|---|
| 1 | **Hakukone** | Google.fi — kokeile hakua `"verkkokauppa" "nuoret" site:stat.fi` tai vastaava | |
| 2 | **Virallinen tilasto** | [Tilastokeskus](https://www.stat.fi) tai [Kaupan liitto](https://kauppa.fi) | |
| 3 | **Sosiaalinen media tai blogi** | Etsi esim. Instagram, TikTok tai toimialan blogi | |
| 4 | **Tekoälytyökalu** | Microsoft Copilot (copilot.microsoft.com) tai ChatGPT | |

> 💡 **Vinkki hakuun:** Hakukoneessa voit rajata tuloksia kirjoittamalla `site:stat.fi verkkokauppa` tai lisäämällä vuosiluvun `verkkokauppa Suomi 2024`.

---

### Vaihe 2 — Lähdekritiikki

Arvioi **jokainen lähde** seuraavien kysymysten avulla. Täytä yksi rivi per lähde.

| Lähde | Kuka julkaisi? | Milloin? | Miksi julkaistu? | Onko luotettava? Miksi / miksi ei? |
|---|---|---|---|---|
| 1. Hakukone | | | | |
| 2. Virallinen tilasto | | | | |
| 3. Some/blogi | | | | |
| 4. Tekoäly | | | | |

**Erityishuomio tekoälyyn:** Vertaa tekoälyn antamaa vastausta Tilastokeskuksen tai Kaupan liiton tietoon.
- Oliko tekoälyn vastaus oikea? ☐ Kyllä ☐ Osittain ☐ Ei
- Jos ei — mitä tekoäly sanoi väärin tai puutteellisesti?

---

### Vaihe 3 — Vastaus johtajalle

Kirjoita lyhyt vastaus (noin 150–250 sanaa) johtajalle. Muotoile se ammatilliseksi sähköpostiviestiksi ja sisällytä:

1. **Vastaus väitteeseen** — onko se totta, osittain totta vai harhaanjohtava?
2. **Perustelu** — mihin lähteisiin vastaus perustuu?
3. **Suositus** — mitä johtajan kannattaa ottaa huomioon palaverissa?

Kirjoita sähköpostiviesti tähän:

> Aihe: Selvitys nuorten ostokäyttäytymisestä
>
> _[Kirjoita vastauksesi tähän]_

---

### Hyväksymiskriteerit

- ✅ Kaikki neljä lähdettä käytetty ja kirjattu
- ✅ Jokainen lähde arvioitu kritiikkitaulukossa
- ✅ Tekoälyn vastaus verrattu muihin lähteisiin
- ✅ Johtajalle kirjoitettu ammatillinen sähköpostiviesti perusteluineen

## Tehtävä 2.2 — Tekoäly tutuksi -työpaja ⭐ *Arvioitava 1–5*

**Arviointi:** 1–5 (vaikuttaa kurssin arvosanaan)  
**Suoritusmuoto:** Ryhmätyö, 2–4 henkilöä  
**Palautus:** Ryhmäesitys (5–10 min) + täytetty vertailupohja eKampukseen

---

### Tilanne

Yrityksenne on harkitsemassa tekoälytyökalujen käyttöönottoa. Johtoryhmä haluaa tietää: **mitkä tekoälytyökalut ovat hyödyllisimpiä, mitkä ovat niiden heikkoudet ja mitä riskejä käyttöön liittyy?** Teidän tiimillänne on yksi oppitunti aikaa selvittää asia käytännössä.

---

### Vaihe 1 — Valmistelu (10 min)

1. Valitkaa ryhmälle **yksi yhteinen toimiala tai aihe**, esim.:
   - Kahvila / ravintola
   - Parturi tai kauneushoitola
   - Urheilukauppa
   - Autokorjaamo
   - Jokin muu, joka tuntuu ryhmältä tutulta

2. Valitkaa **2–3 tekoälytyökalua**, joita kokeilette. Vähintään yksi tekstityökalu vaaditaan:

   | Työkalu | Osoite | Valittu? |
   |---|---|---|
   | Microsoft Copilot | copilot.microsoft.com | ☐ |
   | ChatGPT | chatgpt.com | ☐ |
   | Google Gemini | gemini.google.com | ☐ |
   | Copilot Image Creator (kuvat) | designer.microsoft.com | ☐ |
   | Muu: ____________ | | ☐ |

---

### Vaihe 2 — Kokeilut (25 min)

Antakaa **kaikille valitsemillenne tekstityökaluille täsmälleen sama tehtävä** ja kirjatkaa tulokset.

#### Kokeilu A — Markkinointiteksti

Anna jokaiselle valitsemallesi tekstityökalulle tämä pyyntö (muokkaa lihavoitu osa omaan toimialaanne sopivaksi):

> *"Kirjoita lyhyt ja houkutteleva markkinointiteksti **[toimialanne yrityksen nimi, esim. Café Aurinko]** sosiaaliseen mediaan. Teksti on tarkoitettu Instagramiin. Lisää sopivat hashtagit."*

Kirjatkaa tulokset:

| Työkalu | Kopioi tulos tähän lyhennettynä (tai liitä kuvakaappaus) | Arvosana 1–5 | Perustelut |
|---|---|---|---|
| Työkalu 1: | | | |
| Työkalu 2: | | | |
| Työkalu 3 (jos valittu): | | | |

#### Kokeilu B — Tosiasioiden tarkistus (hallusinaatiot)

Anna jokaiselle tekstityökalulle:

> *"Kertoisitko kolme faktaa [toimialastanne tai ammatistanne], jotka jokaisen alalle tulevan tulisi tietää?"*

Tarkistakaa vastauksista **vähintään kaksi väitettä** hakukoneen tai virallisen lähteen avulla.

| Työkalu | Väite, jonka tarkistitte | Oliko oikein? | Lähde, josta tarkistitte |
|---|---|---|---|
| Työkalu 1: | | ☐ Kyllä ☐ Ei ☐ Osittain | |
| Työkalu 2: | | ☐ Kyllä ☐ Ei ☐ Osittain | |

#### Kokeilu C — Promptin voima

Antakaa **samalle tekoälylle sama pyyntö kahdella eri tavalla** ja vertailkaa eroa:

- **Pyyntö 1 (epämääräinen):** *"Tee meille mainos."*
- **Pyyntö 2 (tarkka):** *"Kirjoita alle 50 sanan mainos [toimialanne yritykselle], jonka kohderyhmä on 20–35-vuotiaat, tyyli on rento ja hauska, ja joka päättyy toimintakehotukseen."*

Mitä eroa tuloksissa oli? Kirjoita havainto:

> _[Kirjoita havaintonne tähän]_

#### Kokeilu D — Kuvageneraattori *(jos valitsitte kuvatyökalun)*

Anna Copilot Image Creatorille tai vastaavalle:

> *"Luo kuva [toimialanne yrityksestä] aurinkoisena päivänä, tunnelma on lämmin ja kutsuva."*

- Miltä tulos näytti? Oliko se käyttökelpoinen? Miksi / miksi ei?
- Kuka omistaa tekoälyn luoman kuvan — yritys, tekoälyn kehittäjä vai kukaan?

---

### Vaihe 3 — Riskianalyysi (10 min)

Pohtikaa ryhmässä ja kirjatkaa vastaukset:

**1. Mitä tietoja ei kannata syöttää tekoälyyn?**  
Esim. asiakkaiden nimet, salasanat... Listatkaa vähintään 3 asiaa:
- 
- 
- 

**2. Milloin tekoälyn vastaus voi olla vaarallinen tai harhaanjohtava?**

> _[Kirjoita vastauksenne tähän]_

**3. Jos käytätte tekoälyä työssänne, miten varmistaisitte, että tuotos on laadukas?**

> _[Kirjoita vastauksenne tähän]_

---

### Vaihe 4 — Esitys ryhmälle (5–10 min)

Valmistelkaa lyhyt esitys (diaesitys tai suullinen + fläppitaulu), jossa kerrotte muulle ryhmälle:

1. Mitä toimialaa/aihetta käytitte ja miksi
2. Mitä tekoälytyökaluja kokeilitte
3. **Paras löytö:** mikä yllätti positiivisesti?
4. **Suurin pettymys tai riski:** mikä ei toiminut tai huoletti?
5. **Suosituksenne:** kannattaako yritysten ottaa tekoäly käyttöön — millä ehdoilla?

> 💡 Esityksessä kaikkien ryhmän jäsenten tulee puhua. Näyttäkää konkreettisia esimerkkejä — älkää vain luetteleko pointteja.

---

### Arviointikriteerit

| Kriteeri | 1 — Tyydyttävä | 3 — Hyvä | 5 — Kiitettävä |
|---|---|---|---|
| **Tekoälyn toimintaperiaatteiden ymmärrys** | Pintapuolinen käsitys, ei perusteluja | Perustoimintaperiaatteet selitetty ymmärrettävästi | Syvällinen ymmärrys; osaa selittää hallusinaatiot, promptin merkityksen ja datan rajoitukset omin sanoin |
| **Vertailu ja analyysi** | Yksi työkalu tai suppea vertailu | Vähintään 2 työkalua vertailtu, perusteltu selkeästi | Monipuolinen, kriittinen analyysi; konkreettiset esimerkit, perustelut vahvoja |
| **Riskien tunnistaminen** | Mainittu 1–2 yleistä riskiä | Riskit tunnistettu ja selitetty käytännön esimerkein | Riskit analysoitu syvällisesti; konkreettisia ratkaisuehdotuksia; yksityisyys ja tekijänoikeudet huomioitu |
| **Esitys** | Luetaan muistiinpanoista, ei vuorovaikutusta | Selkeä esitys, ryhmä osallistuu tasapuolisesti | Vakuuttava, havainnollistettu esimerkein, vuorovaikutteinen — yleisö aktivoitu |

## Tehtävä 2.3 — Tekijänoikeudet käytännössä

**Arviointi:** Hyväksytty / Hylätty  
**Suoritusmuoto:** Yksilötyö  
**Palautus:** Valmis dokumentti + tekijänoikeustaulukko eKampukseen

---

### Tilanne

Yrityksenne tarvitsee pienen **esitteen tai tiedotteen** (1–2 sivua), joka kertoo yrityksestä tai sen palveluista. Sinä olet vastuussa sen tekemisestä. Voit käyttää internetistä löytyviä kuvia ja tekstejä — mutta vain laillisesti ja asianmukaisesti lähdeviitaten.

---

### Mitä teet

Luo **Word-dokumentti tai PowerPoint-esitys** (1–2 sivua / 2–4 diaa), joka sisältää:

- Otsikko ja lyhyt kuvaus yrityksestä tai palvelusta (voit keksiä yrityksen)
- **Vähintään 3 kuvaa**, jotka olet hankkinut laillisesti (ks. Vaihe 1)
- Lyhyttä tekstisisältöä, jossa viitataan lähteisiin asianmukaisesti

---

### Vaihe 1 — Kuvien hankinta

Etsi kaikki kuvasi seuraavista **ilmaisista ja vapaasti käytettävistä** kuvapalveluista:

| Palvelu | Osoite | Huom |
|---|---|---|
| **Unsplash** | unsplash.com | Vapaa käyttö, lähde mainittava (hyvä tapa) |
| **Pixabay** | pixabay.com | Vapaa käyttö, ei vaadi viittausta — mutta hyvä tapa on silti viitata |
| **Pexels** | pexels.com | Vapaa käyttö, lähde mainittava |
| **Wikimedia Commons** | commons.wikimedia.org | Tarkista lisenssi jokaisen kuvan kohdalta! |

> ⛔ **Älä** käytä kuvia suoraan Google-kuvahausta — ne voivat olla tekijänoikeussuojattuja. Google-kuvahaussa voit kuitenkin suodattaa: *Työkalut → Käyttöoikeudet → Creative Commons -lisenssit*.

---

### Vaihe 2 — Tekijänoikeustaulukko

Ennen kuin lisäät kuvan dokumenttiisi, täytä siitä rivi taulukkoon. **Palauta tämä taulukko dokumentin mukana.**

| # | Kuvan kuvaus | Mistä lähteestä? (URL) | Tekijä (jos ilmoitettu) | Lisenssi | Miten käytät kuvaa dokumentissasi? |
|---|---|---|---|---|---|
| 1 | | | | | |
| 2 | | | | | |
| 3 | | | | | |

> 💡 **Miten tunnistat lisenssin?**  
> - Unsplash: Kaikki kuvat ovat Unsplash-lisenssillä (vapaa käyttö)  
> - Pixabay: Tarkista kuvan sivulta "Pixabay License" tai "CC0"  
> - Wikimedia Commons: Jokaisen kuvan sivulla näkyy lisenssi (esim. CC BY-SA 4.0)  
> - CC0 = täysin vapaa käyttö, ei vaadi viittausta  
> - CC BY = voit käyttää, mutta tekijä mainittava  
> - CC BY-NC = voit käyttää, mutta ei kaupalliseen käyttöön  
> - CC BY-ND = voit käyttää, mutta et saa muokata

---

### Vaihe 3 — Tekstilähteet ja tekoäly

Jos käytät dokumentissasi **muiden tuottamaa tekstiä** (esim. tilastoja, faktoja, sitaatteja):
- Lisää lähdeviittaus sulkuihin heti tekstin perään: *(Tilastokeskus, 2024)*
- Lisää lähde dokumentin loppuun *Lähteet*-osioon

Jos käytit **tekoälyä** tekstin tuottamisessa tai muokkaamisessa:
- Mainitse se dokumentin lopussa, esim.: *"Tekstin muotoilussa käytettiin apuna Microsoft Copilotia (4/2026)."*

---

### Vaihe 4 — Omatarkistus ennen palautusta

Käy läpi alla oleva tarkistuslista ennen kuin palautat. Merkitse jokainen kohta tehdyksi.

- ☐ Kaikki kuvat on hankittu laillisesta lähteestä (ei Google-kuvahaun satunnaistulos)
- ☐ Jokaisesta kuvasta on täytetty rivi tekijänoikeustaulukkoon
- ☐ Jokaisen kuvan lisenssi on tarkistettu lähdesivulta
- ☐ CC BY -kuvien tekijä on mainittu dokumentissa tai sen lopussa
- ☐ Tekstilähteet on viitattu
- ☐ Tekoälyn käyttö on mainittu (jos käytetty)
- ☐ Tekijänoikeustaulukko on liitetty palautukseen

---

### Hyväksymiskriteerit

- ✅ Dokumentti sisältää vähintään 3 kuvaa
- ✅ Kaikki kuvat on hankittu luvallisesta lähteestä (ilmaispalvelut tai omat kuvat)
- ✅ Tekijänoikeustaulukko täytetty jokaiselle kuvalle
- ✅ Tekstilähteet viitattu
- ✅ Tekoälyn käyttö mainittu (jos käytetty)
