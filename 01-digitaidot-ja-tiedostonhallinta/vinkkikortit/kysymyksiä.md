## 1. Hallitse tiedostosi (kuvake: **paperiliitin**)

* Nimeä tiedostot selkeästi (esim. `2026-05-raportti.xlsx`)
* Tallenna suoraan OneDriveen ja pidä AutoSave päällä
* Käytä selkeää kansiorakennetta (max 2–3 tasoa)
* Lisää päivämäärä alkuun muodossa `VVVV-KK-PP`, niin tiedostot järjestyvät oikein
* Merkitse versiot tiedostonimeen (`v1`, `v2`, `hyvaksytty`)
* Pidä aktiiviset ja arkistoidut työt erillään (`01_Tarkeat`, `Z_Arkisto`)
* Hae tiedosto nopeasti (**Win + S**)
* Kopioi ja liitä tehokkaasti (**Ctrl + C / Ctrl + V**)
* Siirrä tiedosto leikkaamalla (**Ctrl + X**)
* Nimeä tiedosto uudelleen nopeasti (**F2**)
* Poista turhat tiedostot (**Delete**, pysyvästi **Shift + Delete**)
* Tarkista tiedostopääte ennen avaamista (ole varovainen: `.exe`, `.bat`, `.cmd`, `.xlsm`)
* Käytä PDF:ää, kun lähetät valmiin version eteenpäin

***

## 2. Tee ja muokkaa tehokkaasti (kuvake: **salama**)

* Tallenna työ usein (**Ctrl + S**)
* Kumoa ja tee uudelleen (**Ctrl + Z / Ctrl + Y**)
* Kopioi, leikkaa ja liitä (**Ctrl + C / Ctrl + X / Ctrl + V**)
* Valitse kaikki kerralla (**Ctrl + A**)
* Excel: Täytä sarjoja vetämällä täyttökahvasta
* Excel: Siirry nopeasti alueen reunaan (**Ctrl + nuolinäppäin**)
* Excel: Muotoile data taulukoksi (**Ctrl + T**) ennen analysointia
* Excel: Lukitse vakioarvo kaavassa (`$B$3`), kun kopioit kaavaa
* Excel: Lisää uusi rivi Enterillä taulukon lopussa
* Word: Käytä otsikkotyylejä (**Ctrl + Alt + 1 / 2 / 3**)
* Word: Tee uusi sivu aina sivunvaihdolla (**Ctrl + Enter**)
* Word: Älä muotoile välilyönneillä tai ylimääräisillä Entereillä
* Word: Lihavoi ja kursivoi (**Ctrl + B / Ctrl + I**)
* PowerPoint: Lisää uusi dia (**Ctrl + M**)
* PowerPoint: Käynnistä esitys (**F5**)
* PowerPoint: Pidä yhdellä dialla yksi pääajatus
* PowerPoint: Käytä korkeintaan 2 fonttia ja 2–3 väriä

***

## 3. Jaa ja tee yhdessä (kuvake: **pilvi**)

* Jaa tiedosto linkillä liitteiden sijaan
* Lisää linkki nopeasti (**Ctrl + K**)
* Valitse jakoon oikea taso (katselu tai muokkaus)
* Jaa mieluummin nimetyille henkilöille kuin "kaikille linkin saaneille"
* Muokkaa samaa tiedostoa yhtä aikaa muiden kanssa
* Lisää kommentti yhteistyöhön (**Ctrl + Alt + M**)
* Etsi sisältö nopeasti (**Ctrl + F**)
* Tarkista ennen jakamista kenelle tiedosto näkyy
* Tarkista OneDriven versiohistoria, jos sisältö katoaa tai muuttuu vahingossa
* Sulje tiedosto, kun et enää muokkaa, jotta muiden työskentely pysyy selkeänä
